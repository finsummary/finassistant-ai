# Исправление "AI categorized 0 transactions"

## Проблема

При нажатии "AI Categorize" появляется сообщение "AI categorized 0 transactions".

Это означает, что **нет транзакций для категоризации** - либо:
1. Все транзакции уже категоризированы
2. Транзакции не соответствуют условиям "uncategorized"

## Проверка

### 1. Есть ли транзакции вообще?

**Проверка:**
1. Откройте Dashboard
2. Проверьте список транзакций
3. Если транзакций нет - сначала загрузите CSV

### 2. Проверьте категории транзакций

**Проверка:**
1. Откройте Dashboard
2. Найдите транзакции в списке
3. Посмотрите на поле "Category" для каждой транзакции
4. Что там написано?
   - Пустое поле?
   - "Uncategorized"?
   - Другая категория?

### 3. AI Categorize ищет транзакции с:

- `category` = `null` (пустое)
- `category` = `''` (пустая строка)
- `category` = `'Uncategorized'` (разные варианты написания)
- `category` = `'uncategorized'`
- `category` = `'UNCATEGORIZED'`

**Если ваши транзакции имеют другую категорию** - они не будут обработаны AI Categorize.

## Решение

### Вариант 1: Очистить категории у транзакций

Если транзакции имеют категории, но вы хотите пере-категоризировать:

1. **Вручную:** на Dashboard выберите "Uncategorized" для каждой транзакции
2. **Или через SQL:** (если нужно массово очистить категории)

### Вариант 2: Проверить транзакции

**Убедитесь, что:**
1. ✅ Транзакции загружены (CSV импортирован)
2. ✅ У транзакций поле `category` пустое или "Uncategorized"

### Вариант 3: Использовать ручную категоризацию

**Если AI Categorize показывает 0:**
1. Назначьте категории вручную через выпадающий список
2. Это работает мгновенно и не требует AI

## Проверка транзакций в базе

Если хотите проверить транзакции напрямую:

1. **Supabase Dashboard:**
   - Database → Table Editor → `Transactions`
   - Проверьте колонку `category`
   - Какие значения там?

## Ожидаемое поведение

**AI Categorize должен найти транзакции, если:**
- Транзакции существуют в базе
- Поле `category` пустое (`null` или `''`)
- Или `category` = "Uncategorized"

**Если все транзакции уже имеют категории** - сообщение "0 transactions" нормально.

## Что делать дальше?

1. **Проверьте транзакции на Dashboard:**
   - Есть ли транзакции?
   - Какие у них категории?

2. **Если транзакции есть, но категории не "Uncategorized":**
   - Используйте ручную категоризацию
   - Или очистите категории и попробуйте AI Categorize снова

3. **Если транзакций нет:**
   - Загрузите CSV файл через Dashboard → Import CSV
